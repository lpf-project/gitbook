# 其他插件

### tbfed-pagefooter

在 book.json 中配置 tbfed-pagefooter 插件,详细说明请参考 tbfed-pagefooter 插件.

```json
{ 
    "plugins": ["tbfed-pagefooter"], 
    "pluginsConfig": { 
        "tbfed-pagefooter": { 
            "copyright":"&copy snowdreams1006", 
            "modify_label": "文件修订时间：", 
            "modify_format": "YYYY-MM-DD HH:mm:ss" 
        } 
    } 
}
```
